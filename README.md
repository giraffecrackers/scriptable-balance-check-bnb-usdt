# Scriptable app widget to check USDT balance on BNB chain

## Simple guide

- Install [Scriptable app](https://scriptable.app)
- Create script inside of the app
- Copy content of `script.js` into created script
- Click 'Done'
- Add scriptable widget to the sceen you want
- During widget setup set parameter to the wallet address you want to see balance for
